# CentOS Image generator (and downloader)

Generates auto-install ISOs for CentOS. May also work for RedHat.

Usge example:

    ./createiso.sh /tmp/CentOS-7-x86_64-NetInstall-2003.iso /home/user/ks/minimal-7.ks.cfg /home/user/isos/
